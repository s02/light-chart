# Варианты билдов
* standalone демо-приложение, работающее с qa
* vue-компонента, позволяющая непосредственно подключить к терминалу. Сам vue используется как peer dependency, т.е. в билде его нет.
* vue-приложение, позволяющее интегрировать в любое другое приложение (например во флаттер)
* сам engine (опционально)
